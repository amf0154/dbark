# InfluxDB Backup System - Обновления

## Добавленные функции (аналогично MySQL)

### 1. Расширенные функции очистки

#### Новые команды в manage.sh:

```bash
# Удаление локальных файлов по типам
./scripts/influxdb/manage.sh clean-local backups      # Только бэкапы
./scripts/influxdb/manage.sh clean-local pre-restore  # Только pre-restore файлы
./scripts/influxdb/manage.sh clean-local all          # Все локальные файлы

# Управление расписанием бэкапов
./scripts/influxdb/manage.sh schedule show            # Показать текущее расписание
./scripts/influxdb/manage.sh schedule set 'CRON'     # Установить новое расписание
./scripts/influxdb/manage.sh schedule disable        # Отключить автобэкапы
./scripts/influxdb/manage.sh schedule examples       # Примеры расписаний

# Синхронизация с Google Drive
./scripts/influxdb/manage.sh sync                     # Загрузить бэкапы в Google Drive
```

## Использование

### Быстрый старт с новыми функциями:

```bash
# 1. Обновить конфигурацию
./scripts/influxdb/setup.sh

# 2. Запустить систему
./scripts/influxdb/manage.sh start

# 3. Настроить расписание (каждые 6 часов)
./scripts/influxdb/manage.sh schedule set '0 */6 * * *'

# 4. Очистить старые файлы
./scripts/influxdb/manage.sh clean-local pre-restore

# 5. Синхронизировать с Google Drive
./scripts/influxdb/manage.sh sync

# 6. Проверить здоровье системы
./scripts/influxdb/manage.sh health
```
